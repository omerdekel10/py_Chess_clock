# CHESS CLOCK #

A simple script that generates a GUI chess clock/

How to use:
Input minutes and hours in whole numbers. Press 'Start game' when ready, clock will count down right away. Restart button brings players back to input.

Modules:
Datetime creates the time frame and timer.
Tkinter creates all of the windows - time input, info, timer, win.
